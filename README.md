# 📈 Telegram Stock Analysis Bot + Mini App

A full-stack stock analysis and trading assistant with both a Telegram bot and a Telegram Mini App interface.

## 🚀 Features

- 📊 Technical analysis (RSI, MA, MACD)
- 🧠 Score-based system (0–100)
- 🔍 Multi-symbol scan
- 🏷️ Opportunity tags
- 💰 Buy / Sell simulation
- 📁 Portfolio tracking
- ⭐ Watchlist system

## 🧱 Tech Stack

- Backend: FastAPI (Python)
- Frontend: React (Vite)
- Bot: python-telegram-bot
